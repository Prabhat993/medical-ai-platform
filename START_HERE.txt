STEP 1: INSTALL DEPENDENCIES
-----------------------------
Open terminal in this folder and run:

    pip install -r requirements.txt

This installs: Flask, TensorFlow, NumPy, Pillow

STEP 2: DOWNLOAD DATASET (Required for training)
------------------------------------------------
1. Go to: https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia
2. Download the dataset (requires free Kaggle account)
3. Extract the ZIP file
4. Copy these folders to your project root:
   - train/  (from the extracted chest_xray folder)
   - test/   (from the extracted chest_xray folder)

Your folder should look like:
ai_medical_platform/

STEP 3: TRAIN THE AI MODEL
---------------------------
Run this command (will take 30 minutes to several hours):

    python train_model.py

Wait for it to complete. It will create:
    model/pneumonia_model.h5

STEP 4: RUN THE APPLICATION
----------------------------
Start the web server:

    python app.py

You should see:
    Loading AI Model... please wait.
    Model loaded successfully!
    * Running on http://127.0.0.1:5000

STEP 5: USE THE APPLICATION
----------------------------
1. Open your browser
2. Go to: http://127.0.0.1:5000
3. Upload an X-ray image (test images in test/NORMAL or test/PNEUMONIA)
4. Click "Analyze X-Ray"
5. View AI prediction with confidence score
